Wat heb ik voor mijn opleiding gedaan?

Voor de opleiding zat ik nog op de middelbare school (OSB). De eerste 2 jaren heb ik VMBO-T gevolgd en de laatste 2 jaren VMBO-K. In de laatste 2 jaren heb ik de richting (PIE) Produceren Installeren en Energie gevolgd.

PIE

Bij mijn richting PIE heb ik geleerd hoe ik een schakeling kan maken, metaal kan bewerken, met hout kan werken en met relais kan omgaan. Voor mijn eindexamen van PIE moest ik een schakeling maken op een bord, een laptopstandaard maken, en een badkamer ontwerpen.

Software Developer

Ik ben begin dit schooljaar begonnen met Software Developer. Ik heb de basis van programmeertalen geleerd zoals HTML, C#/C++, PHP en JavaScript.
